# Q005 - Maximum Seating Capacity Of Events

## Problem

The event manager wants to find the maximum seating capacity for each event with WiFi access.

Write a query to display the event id and maximum seating capacity for all events.

Give the alias name to the maximum seating capacity as **maximum_seating_capacity**.

Sort the results based on **maximum_seating_capacity** in descending order.

**Hint:** WiFi values are **'Yes'** or **'No'**.
