# Q001 - Customer Password

## Problem

Write a query to display the first name and password of all the customers with 'gmail' accounts.

The password must contain:
- First 3 characters of the customer's first name
- Followed by the 5th and 6th digits of the customer's phone number

Give aliases:
- user_name to customer's first name
- password to customer's password

Sort the results by user_name in ascending order.

Follow the table naming convention as is the "Reference Table"
