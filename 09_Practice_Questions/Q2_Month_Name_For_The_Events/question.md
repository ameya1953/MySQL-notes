# Q002 - Month Name For The Events

## Problem

Write a query to display the enquiry_id, cust_id and Event_Date for all enquiries made after the year 2015.

Requirements:
- Display Event_Date in the format MonthName,Year.
- Give the alias name as Event_Date for the event start date.
- Sort the results by Event_Date in ascending order and cust_id in ascending order.

Example Output:
April,2017
April,2018
