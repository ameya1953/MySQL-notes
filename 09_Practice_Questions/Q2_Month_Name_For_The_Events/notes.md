# Notes

## Functions Used

### MONTHNAME()

Returns the month's name.

Example:

MONTHNAME(start_date)

Output:

April

---

### YEAR()

Returns the year.

Example:

YEAR(start_date)

Output:

2017

---

### CONCAT()

Joins multiple strings.

Example:

CONCAT(MONTHNAME(start_date), ',', YEAR(start_date))

Output:

April,2017

---

### ORDER BY

ORDER BY Event_Date ASC, cust_id ASC

Sorts by:
1. Event_Date
2. cust_id
