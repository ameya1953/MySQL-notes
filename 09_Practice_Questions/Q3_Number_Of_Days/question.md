# Q003 - Number Of Days

## Problem

Write a query to display the enquiry id, customer id, venue id, month name of the event starting date with the alias as month_name, and the number of days the event has been enquired about the venue with the alias name as no_of_days for the enquiries made in the month of December.

Give the alias:
- month_name as month name of the starting date
- no_of_days for the enquiries made in the month of December

Sort the results by no_of_days in descending order.
